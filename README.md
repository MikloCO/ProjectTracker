## Why
I encountered a very unique problem. I love learning new things, I want to do art, vfx and programming courses. But Because these diciplines are so different, there are very few platforms that cover both topics. So I have many courses across different websites (gumroad, skillshare, udemy, wingfox) etc. That I want to do. And sometimes, I even forget if I have important courses somewhere waiting for me... 

So I wanted to give the opportunity for other people who struggle with the same issue: to be able to track their progress in different courses from different websites in the same tool!


## What
Project tracker is your centralized place where you can access, and save information about all the courses from!

Key features:

- Track all your courses across different platforms
- Keep notes and important links in one place
- Record where project files are stored on your machine
- Never forget unfinished or valuable courses again
- Image gallery per course (users can save screenshots/work images to each course entry)
- Chapter-by-chapter progress tracking

You can now: remember which total amount of courses you have, where on your machine you have the project, any notes
and which website things are on.
